# Daily Routine Optimization

**Live Demo:** https://planner-5u1tba3v7-ashrii04s-projects.vercel.app/

## Overview
The **Daily Routine Optimization** platform is a full-stack application designed to help users manage, organize, and optimize their daily tasks. It suggests the best sequence of tasks, tracks progress, and adapts routines for maximum efficiency, enhancing productivity and time management.

---

## Features
- **Task Prioritization:** Automatically organizes tasks based on urgency and importance.  
- **Time Management:** Suggests optimal time slots to complete tasks.  
- **Analytics Dashboard:** Tracks task completion patterns and productivity metrics.  
- **Adaptive Recommendations:** Learns user habits and improves routine suggestions.  
- **Responsive Design:** Works seamlessly on desktop and mobile browsers.

---

## Tech Stack
- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Deployment:** Vercel (Frontend), Node.js/Express server (Backend)  
- **Other:** Axios for API requests, CSS for styling
